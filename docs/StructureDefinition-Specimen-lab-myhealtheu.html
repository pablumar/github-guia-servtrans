Resource Profile: Specimen: Laboratory

Official URL: http://fhir.ehdsi.eu/laboratory/StructureDefinition/Specimen-lab-myhealtheu Version: 0.0.1
Draft as of 2025-03-03 Computable Name: SpecimenMyHealthEu

This profile defines how to represent Specimens in HL7 FHIR for the purpose of this guide.

Usage:

  • Use this Resource Profile: Bundle: Laboratory
  • Refer to this Resource Profile: DiagnosticReport: Laboratory, Observation Results: Laboratory, ServiceRequest: Laboratory and Specimen: Laboratory
  • Examples for this Resource Profile: Specimen/SampleSpecimen and Specimen/Specimen-device-example
